# ng2-base
Setup for Angular 2 release candidate, configured to use JSPM and SystemJS.
## To get started
Install jspm and typings globally:
<pre>
    npm install -g jspm
    npm install -g typings
</pre>
Then run the setup:
<pre>
    npm run setup
</pre>
Now start the local web server by:
<pre>
    npm start
</pre>
